**Time Complexity:** $\mathcal{O}(N^2)$
This problem is an extension of the two sum problem except now with three values. We can set the third pointer to a certain value in the array and then the problem becomes the two sum problem discussed earlier in this module. We'll first sort the array, then we'll loop through all possible values for the third pointer and check if all three pointers add up to out target amount, $x$, while having distinct positions. We can maintain the original positions of all values by storing them as a pair with the value being the first value and the position being in the second value/\.

# Solution



```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
using vi = vector<int>;
#define pb push_back
#define rsz resize
#define all(x) begin(x), end(x)
#define sz(x) (int)(x).size()
using pi = pair<int,int>;
#define f first
#define s second
#define mp make_pair
void setIO(string name = "") { // name is nonempty for USACO file I/O

ll dp[1000001];

const int MOD = (int) 1e9 + 7;

int main(){
    int n, x; cin >> n >> x;
    vi coins(n);
    for (int i = 0; i < n; i++) {
        cin >> coins[i];
    }
    dp[0] = 1;
    for (int i = 1; i <= n; i++) {
        for (int weight = 0; weight <= x; weight++) {
            if(weight - coins[i - 1] >= 0) {
                dp[weight] += dp[weight - coins[i - 1]];
                dp[weight] %= MOD;
            }
        }
    }
    cout << dp[x] << '\n';
}
```
